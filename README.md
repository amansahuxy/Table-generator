# 📋 Multiplication Table Generator

A simple Multiplication Table Generator built using HTML and JavaScript. Users can enter a number and generate its multiplication table from **1 to 10**.

## 🚀 Features

* Generate multiplication tables
* Display results in a textarea
* Simple and beginner-friendly interface
* Instant table generation

## 🛠️ Technologies Used

* HTML5
* JavaScript

## 📁 Project Structure

```text
Table-Generator/
│
├── index.html
├── index.js
└── README.md
```

## ▶️ How to Run

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## 🧠 How It Works

1. Enter a number in the input field.
2. Click the **Generate Table** button.
3. The application generates the multiplication table from **1 to 10**.
4. The table is displayed inside the textarea.
